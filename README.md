vatcalc
=======

Few very basic functions to work with prices in VAT based countries:

* net_to_gross
* gross_to_net
* vat_amount_from_net
* vat_amount_from_gross
* discount_percent
* discount_amount

Too many times have I copied one or more of these from one project to another. Enough is enough.

Installation
------------

    uv add vatcalc
